## 判分规则

| 输入 | 正确判断 |
| --- | --- |
| `001 -> 101` | 合法相邻跳变，计一次有效边沿，不算异常，方向只能叫候选。 |
| `001 -> 001` | 重复状态，不计新边沿，不直接算异常，可记录重复。 |
| `001 -> 010` | 两个都是有效状态，但不是当前候选序列中的相邻跳变，计异常跳变，不当正常旋转。 |
| `000` | 非法 Hall 状态，不计边沿，不更新角度/速度，计非法状态。 |
