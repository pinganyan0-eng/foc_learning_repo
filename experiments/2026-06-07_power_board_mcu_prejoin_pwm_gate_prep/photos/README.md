## Photo Rules

- 原图保留，不用聊天软件压缩版本替代原图。
- 一张照片只承担一个主要证据目标。
- 测点可用非导电标记或后期箭头标注，不把金属物放在通电板附近。
- 文件旁附同名 `.md` 时，记录来源、拍摄时间、板卡版本和照片能证明/不能证明的结论。
