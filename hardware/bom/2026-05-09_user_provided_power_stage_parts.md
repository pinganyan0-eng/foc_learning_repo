## 目标电流与保护阈值

| 参数 | 当前记录 |
| --- | --- |
| 电机型号 | 42BLF01-2430HE |
| 电机额定电流 | 1.2A，用户称为官方标称 |
| 电机峰值电流 | 3A，短时堵转/加速工况 |
| 软件限流阈值 | 8A，STM32 ADC 采样触发 |
| 硬件 VDS 过流保护目标值 | 约 55A，STDRIVE101 内部比较器触发 |
| 用户给出的计算依据 | SCREF 分压 33kΩ + 20kΩ -> V_SCREF = 1.245V -> V_DSth = 0.249V；NCEP40T11G Rds(on)=4.5mΩ；I_trip = 0.249V / 0.0045Ω ≈ 55A |
| 复核提醒 | 55A 远高于 1.2A 额定电流、3A 峰值电流和 8A 软件限流，更像短路级硬保护目标；必须按 STDRIVE101 Datasheet、MOS 热态 Rds(on) 和实际 SCREF/VDS 关系复核。 |
